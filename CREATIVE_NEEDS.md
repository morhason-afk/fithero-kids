## ðŸŽ¨ **GLOBAL DESIGN SYSTEM**

### **Color Palette**
- **Primary Colors**:
  - Main brand color: Vibrant blue/purple gradient (#667eea â†’ #764ba2)
  - Success: Green (#4ade80, #22c55e)
  - Warning: Yellow/Orange (#fbbf24, #f59e0b)
  - Error: Red (#ef4444)
  - Info: Blue (#2196f3, #64b5f6)
- **Neutral Colors**:
  - Background: Light gray/white (#ffffff, #f5f5f5)
  - Text: Dark gray/black (#333333, #000000)
  - Borders: Light gray (#e0e0e0)
- **Gradient Usage**: 
  - Buttons, cards, backgrounds
  - Smooth transitions

### **Typography**
- **Headings**: 
  - Bold, rounded sans-serif (e.g., Nunito, Poppins, Quicksand)
  - Size hierarchy: H1 (2.5rem), H2 (2rem), H3 (1.5rem)
- **Body Text**: 
  - Readable sans-serif
  - Size: 1rem (16px) base, 0.875rem (14px) secondary
- **Special Text**:
  - Numbers: Monospace or bold
  - Labels: Smaller, uppercase or lowercase with tracking

### **Spacing & Layout**
- **Grid System**: 
  - 8px base unit
  - Consistent padding/margins
- **Card Spacing**: 
  - 16-24px padding
  - 16-24px gaps between cards
- **Section Spacing**: 
  - 32-48px between major sections

### **Icons & Illustrations**
- **Style**: 
  - Consistent style (outline or filled)
  - Kid-friendly, rounded corners
  - Size: 24px, 32px, 48px variants
- **Illustrations**:
  - Custom character illustrations
  - Challenge type icons
  - Decorative elements (stars, sparkles, badges)

### **Animations & Transitions**
- **Duration**: 
  - Quick: 150-200ms (hover, click)
  - Medium: 300-400ms (transitions)
  - Slow: 500-800ms (celebrations, entrances)
- **Easing**: 
  - Ease-out for entrances
  - Ease-in-out for transitions
  - Bounce for celebrations
- **Micro-interactions**:
  - Button press feedback
  - Card hover effects
  - Progress bar animations
  - Coin counter animations

### **Accessibility**
- **Color Contrast**: 
  - WCAG AA compliant (4.5:1 for text)
  - Color not sole indicator
- **Touch Targets**: 
  - Minimum 44x44px
  - Adequate spacing
- **Focus States**: 
  - Visible focus indicators
  - Keyboard navigation support
- **Text Readability**: 
  - Appropriate font sizes
  - Sufficient line height
