# How to Run the Game

## 1. Start the server
In Terminal, run:
```bash
cd /Users/morhasson/Documents/exercise-game-app
npm run dev
```

## 2. Open the correct URL
- If you see **Local: http://localhost:3000** → open **http://localhost:3000**
- If you see **Port 3000 is in use, trying 3001** → open **http://localhost:3001**

Use whichever port the server shows in the terminal.

## 3. If the page is blank or not loading
- Do a **hard refresh**: `Cmd + Shift + R` (Mac) or `Ctrl + Shift + R` (Windows)
- Or open the URL in a **private/incognito** window
- Make sure you're using **http://** (not https) and the correct port
